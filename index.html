Configurar banco de dados compartilhado
Siga os 4 passos abaixo para conectar o Firebase gratuitamente. Feito uma vez, todos os colaboradores compartilharão os mesmos dados em tempo real.
1
Acesse console.firebase.google.com, clique em "Criar projeto", escolha um nome (ex: nuevo-implantacoes) e desative o Google Analytics.
2
No menu lateral do projeto, clique em "Realtime Database" → "Criar banco de dados" → região us-central1 → "Modo de teste" → Ativar.
3
Copie a URL que aparece no topo do banco, no formato:
https://nuevo-implantacoes-default-rtdb.firebaseio.com
4
Cole a URL abaixo e clique em Conectar. Esta configuração é salva automaticamente.
URL inválida. Use o formato: https://PROJETO-default-rtdb.firebaseio.com
